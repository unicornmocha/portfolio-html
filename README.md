# portfolio-html
## Portfolio HTML Website

This is my HTML portfolio website, created as part of a Full Stack Web Development Bootcamp. The website showcases multiple projects I've completed during the course, demonstrating my skills in HTML and basic web development concepts.

Projects Included:

- Birthday Invitation – A simple single page website for sending birthday invitations. Projects aim was to learn about images and hyperlinks.

- Movie List – A movie list website showcasing skills in using basic html elements like h, p, hr.

- Portfolio Website – The main project that links to previous projects and serves as my personal web portfolio.

Features:

- Multi-page navigation between different projects.

- Clean HTML structure and organized folder/file setup.


How to View:

Clone the repository and open index.html in a web browser to view the portfolio and navigate through the projects.
